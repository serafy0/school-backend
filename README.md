# school-backend

a school backend api

## Tech  Stack

  - Node.js
  - Objection
  - knex
  - PostgreSQL
 
## Installation

  - ```npm install```
  - Setup a postgres server and add it to  `db/knexfile.js`
  - `npm start`
  - the app should run in port `8088`
  - install [mailhog](https://github.com/mailhog/MailHog) to catch api's emails locally
